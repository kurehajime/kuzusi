## Download

Download binaries from Releases:
https://github.com/kurehajime/kuzusi/releases

## macOS note

If macOS shows a warning like "Apple cannot verify 'kuzusi' is free of malware", run this before executing:

```sh
xattr -d com.apple.quarantine ./kuzusi
```

## Build

You can build release artifacts (zip for each platform) with:

```sh
make out
```
